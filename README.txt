Run playDetonicon.py from this directory to play Detonicon once.

autoTestingScript_lag.py was used in the bot study (tests one latency with or without latency compensation).
autoTestingScript_lag_Called was used in the boy study (tests a range of latencies with and without latency compensation.

userStudyScript.py was used in the user study.

detonicon/ contains the needed files to be able to play Detonicon
maps/ contains the maps needed to play Detonicon
results/ is the directory where statistics are recorded to
sounds/ contains the sounded needed to play Detonicon
sprites/ contains the sprites needed to play Detonicon
